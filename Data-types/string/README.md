| Function    | Meaning              |
| ----------- | -------------------- |
| `upper()`   | Convert to uppercase |
| `lower()`   | Convert to lowercase |
| `title()`   | First letter capital |
| `strip()`   | Remove spaces        |
| `replace()` | Replace text         |
| `split()`   | Split into list      |
| `len()`     | Count characters     |
